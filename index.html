Introduction

What is NotebookLM

Updated step-by-step guide

From first access to the most advanced features

"NotebookLM does one thing, and it does it well: it takes the materials you already use and turns them into ready-to-use tools. It adds nothing of its own, it does not browse the internet, it does not invent. It only works with what you give it."

1

Accessing and creating a notebook

Go to notebooklm.google.com and click "New notebook". NotebookLM responds only to the sources you provide — this eliminates "hallucinations" and creates a safe information boundary.

2

Managing sources (uploading and searching)

Classic upload: PDF, copied text, websites, YouTube videos. You can also ask it to find files in your Google Drive or activate Deep Research to search the web and build a report as the primary source.

3

Advanced strategy: have it write the prompt for you

One of the most powerful features is using NotebookLM to write instructions for itself: ask "Create a detailed prompt to create slides for…", copy the result and reuse it.

4

Transforming documents into content

Presentations, Video overviews, Audio overviews (podcasts), Concept maps, Infographics, Data tables — all tools generatable in a few clicks.

5

Sharing: personalised chatbot

You can share the notebook in "viewer" mode: whoever receives the link can ask questions but cannot see the original files. It is like creating an expert assistant on your materials.

6

Mobile app and studying on the go

NotebookLM app for iOS and Android with automatic flashcards and quizzes. Study and review even outside the classroom.

The interface

How the interface is structured

Three zones. If you understand these, you know how to use NotebookLM.

1

Left column:
Sources

This is the library: this is where you upload materials. Without sources, the assistant cannot work.

  • + Source: upload PDFs, links, web content, videos.
  • Checkboxes: choose which documents to use now.
  • YouTube: the AI transforms the video into analysable text.

Teacher's tip

Check only the sources useful for that lesson. Fewer sources = more focused answer.

2

Central zone:
Chat

Here you make requests, get answers and check citations.

  • Message bar: write simple requests (e.g. "Create an assessment").
  • Citations [1]: click the numbers to see the source in the document.
  • Save note: keep useful outputs (UDA, summaries, outlines, exercises).

Quality rule

If the answer is generic: reduce checked sources or ask for a more specific format (table, list, levels).

3

Top / right:
Study Zone

Here NotebookLM automatically produces teaching tools from the checked sources.

  • Overview: reasoned overview of content.
  • Concept map: concepts and relationships (very useful for SpLD).
  • Infographic: structured summary, quickly readable.
  • FAQ: automatic frequently asked questions (active review).
  • Study guide: step-by-step path with priorities.
  • Data Table: extracts info and organises it in structured tables.

Key idea

Start from the Study Zone (orientation), then move to Chat (exercise production).

Study Zone

Study Zone: every function explained

Here NotebookLM transforms sources into teaching tools.

1 OVERVIEW (General overview)

The Overview is a reasoned panorama of the entire selected material. It is not a "short summary": it helps understand what the material is about and how it is organised.

What it does for the teacher

  • • quickly understand a long PDF
  • • decide what to explain and what to cut
  • • build an introduction to the lesson

Teaching value

  • • useful for students who "get lost"
  • • basis for guided pre-reading
  • • support for interdisciplinary learning units
2 CONCEPT MAP

The Concept map reorganises content by highlighting concepts and relationships (cause-effect, sequences, hierarchies).

For the teacher

  • • set up a clear explanation
  • • prepare oral tests and assessments
  • • identify critical nodes

Inclusion

  • • reduces cognitive load
  • • facilitates memorisation
  • • useful for SpLD/SEN

Tip

If the map is too long: uncheck secondary sources or work on a single paragraph.

3 INFOGRAPHIC

The Infographic is a summary organised in clear, quickly readable blocks. It is not "a pretty picture": it is an information structure ready to copy into slides or cards.

For the teacher

  • • create the final summary slide
  • • produce review cards
  • • make a dense text accessible

For students

  • • focus on key concepts
  • • quick pre-assessment review
  • • comprehension support
4 FAQ (Frequently asked questions)

The FAQ function generates typical questions with source-based answers. Perfect for active review, remediation and oral exams.

For the teacher

  • • prepare oral tests
  • • create comprehension exercises
  • • anticipate difficulties

Teaching value

  • • stimulates metacognition
  • • reduces study anxiety
  • • great for pair work

Classroom mode

Project the FAQs: students answer first, then compare with the output and the citation.

5 STUDY GUIDE

The Study guide organises material into a path: what to study first, what next, what to focus on. Concrete support for students who do not know "where to start".

For the teacher

  • • remediation and reinforcement
  • • summer study / make-up periods
  • • differentiation (basic/advanced)

For SEN/SpLD

  • • reduces disorientation
  • • creates study routines
  • • helps manage time and priorities

Advanced feature

Deep Research

NotebookLM searches the web for you and builds a reliable source.

How to get there — exact path in the interface

Left column Sources Field "Search new sources on the web" Button 🌐 Web Arrow ▼ next to "Quick search"

The dropdown menu shows two options: Quick search (Fast Research) and In-depth search (Deep Research). It is not a separate button — it is hidden in the dropdown!

What is Deep Research

When you have no available materials on a topic, you can activate Deep Research: NotebookLM sends intelligent agents onto the web, analyses hundreds of sources, synthesises them and creates a structured report that automatically becomes a source in your notebook.

Autonomous agents

The agents browse the web, assess source reliability and collect only relevant information.

Report as source

The result is automatically added as a source in the notebook, ready to be used with all other tools.

Web citations

Every piece of information in the report is tracked with a link to the original source — no hallucinations.

When to use it (school)

Regulatory updates

Are you preparing a lesson on a recent CAP or environmental regulation? Deep Research finds the latest circulars and makes them available as a source.

Current events for Civic Education

For learning units on topics like sustainability, food security or climate change: search for recent articles and build a reliable source.

Technical insights

Drip irrigation techniques, new crop varieties, phytosanitary treatments: get an updated report without searching manually.

Warning

Always review the report before using it with students. Web sources can vary in reliability — your professional oversight remains indispensable.

"Use Deep Research to find updated information on Italian legislation on agricultural waste management. Create a report with: regulatory context, main obligations, penalties, and 3 practical examples for a small farm."
Data structuring

"Data Table" Feature

Transforms scattered information into structured data.

What it is and what it does

The "Data Table" function transforms discursive, scattered or disorganised information into a logical row-and-column format — ready to be exported to Excel or Google Sheets.

1. Targeted extraction

Isolates specific data hidden in long documents (manuals, regulations) and groups them in a single view.

2. Comparison

Compares multiple elements by crossing variables: pros/cons, characteristics, time deadlines.

3. Glossaries and outlines

Automatically generates term lists with definitions, translations or technical characteristics.

4. Export

Output ready for Google Sheets or Microsoft Excel for further processing.

How to activate it

Specify the desired columns in the chat box:

"Extract all procedures cited in the documents and create a table divided into: Procedure name, Main phases, Necessary tools"

Troubleshooting

Common problems and solutions

Before thinking "it does not work", check these points.

Answers too generic

  • • reduce checked sources
  • • request a format (table / list / levels)
  • • narrow the topic (chapter/paragraph)

Missing citations

  • • verify the PDF is selectable text (not a scan)
  • • upload a "text" version of the document
  • • limit sources and try again

Confused / contradictory results

  • • you have sources with different definitions: separate them
  • • work "in blocks": one source at a time
  • • use Overview first to understand what is inside

Too much time spent writing

  • • start from the Study Zone (automatic outputs)
  • • use the universal prompt templates (section above)
  • • save the best answers as reusable notes

PDF not loading

  • • maximum size: 200 MB per file
  • • scanned PDFs (images) may not be read correctly
  • • try converting the PDF with Adobe or an OCR tool
  • • alternatively paste the text directly into the chat

Audio Overview in the wrong language

  • • the audio language follows the language of uploaded sources
  • • for English: upload sources in English
  • • specify the language in the instructions before generating
  • • if mixed: the predominant language wins
